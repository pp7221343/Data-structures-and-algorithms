# 找到旋转数组中的最小值
* 输入：{3，4，5，1，2}
* 输出：1；

思路比较简单,遍历一遍，取第一个数给min，如果遍历到的数比下一个大，就把下一个赋值给min并返回

```java
class Solution {
    public int minArray(int[] numbers) {
        if(numbers.length == 0){
            return 0;
        }
        int min = numbers[0];
        for(int i = 0;i<numbers.length-1;i++){
            if(numbers[i] > numbers[i+1]){
                min = numbers[i+1];
            }
        }
        return min;
    }
}
```
